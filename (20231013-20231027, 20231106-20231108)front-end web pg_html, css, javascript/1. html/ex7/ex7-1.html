배경 이미지 삽입

웹페이지에 배경 화면을 삽입하는 데에는 background-image속성을 이용한다. 배경이미지가 들어갈 영역보다 작을 때에는 배경 이미지가 수평과 수직방향으로 반복된다.





p{ width:600px; line-height:180%; }
body{ background-image:url("https://cdn.imweb.me/upload/S202207202685e30f16e24/3cf57f6774054.png"); background-size:cover; background-repeat: no-repeat; background-position:center center; }
